# highlightjs4tistory
- highlight.js for tistory plugin.

# Installation
```
<link rel="stylesheet" href="//cdnjs.cloudflare.com/ajax/libs/highlight.js/9.3.0/styles/default.min.css">
<script src="//cdnjs.cloudflare.com/ajax/libs/highlight.js/9.3.0/highlight.min.js"></script>
<script>hljs.initHighlightingOnLoad();</script>
<script src="//github.com/Vince-Yi/highlightjs4tistory/blob/master/highlightjs4tistory.js"><script>
```

# Usage
```
<blockquote class="tx-quote-tistory"><p>### html</p>
<p>&lt;!-- cdn 에서 라이브러리 로드&nbsp;--&gt;</p>
<p>&lt;link rel="stylesheet" href="//cdnjs.cloudflare.com/ajax/libs/highlight.js/9.3.0/styles/default.min.css"&gt;</p>
<p>&lt;script src="//cdnjs.cloudflare.com/ajax/libs/highlight.js/9.3.0/highlight.min.js"&gt;&lt;/script&gt;</p></blockquote><p><br /></p><blockquote class="tx-quote-tistory"><p>### html</p>
<p>&lt;pre&gt;&lt;code class="html"&gt;...&lt;/code&gt;&lt;/pre&gt;</p></blockquote>
```
